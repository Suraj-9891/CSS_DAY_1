CSS (Cascading Style Sheets)

What is CSS?

CSS (Cascading Style Sheets) is a styling language used to control the layout and appearance of web pages written in HTML or XML. It separates presentation from content, making it easier to maintain and update websites.

Key Features of CSS:

  1. Selectors: Target specific HTML elements to apply styles.
  2. Properties: Define styles for elements (e.g., color, font-size).
  3. Values: Assign values to properties (e.g., red, 18px).
  4. Classes: Group elements for reusable styles.
  5. ID: Unique identifier for an element.
  6. Inheritance: Styles cascade down from parent elements.
  7. Box Model: Defines element layout (margin, padding, border).

CSS TYPES:

  1. 1. Internal CSS: Styles within an HTML document.
  2. 2. External CSS: Separate CSS file linked to HTML.
  3. 3. Inline CSS: Styles applied directly to HTML elements.

CSS Properties:

  1. Color: Text, background, border colors.
  2. Font: Family, size, style, weight.
  3. Layout: Display, position, float, clear.
  4. Background: Image, color, repeat.
  5. Border: Style, width, color.
  6. Margin: Space between elements.
  7. Padding: Space within elements

CSS Frameworks:

  1. Bootstrap: Popular, responsive design.
  2. Materialize: Google's material design framework.
  3. Tailwind CSS: Utility-first framework.

Benefits of CSS:

  1. Consistency: Uniform styling across website.
  2. Reusability: Easy maintenance and updates.
  3. Flexibility: Responsive design for various devices.
  4. Accessibility: Improved readability and navigation.

Common CSS Tools:

  1. CSS Preprocessors: Sass, Less.
  2. CSS Editors: Visual Studio Code, Sublime Text.
  3. CSS Validators: W3C Validator.
